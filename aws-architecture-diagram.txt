## Data Flow:

1. Developer pushes code to GitHub
2. Build script creates Docker image and pushes to ECR
3. Deployment artifacts uploaded to S3
4. Elastic Beanstalk pulls image from ECR
5. Application deployed on EC2 with ALB
6. App connects to RDS PostgreSQL with SSL
7. CloudWatch monitors logs and metrics

## Key Features:

• Containerized deployment with Docker
• Auto-scaling with Elastic Beanstalk
• Secure database connections (SSL)
• Load balancing with ALB
• Centralized logging with CloudWatch
• Automated CI/CD pipeline
• Bilingual interface (PT/EN)
• Dark/Light theme support
